## js-Excel 
Js is used to operate Excel tables and realize automatic summation.

使用js对在网页中打开Excel表格，并进行自动求和操作。

#### 使用的插件：`js-xlsx`

#### blog地址：[www.caomage.com](http://www.caomage.com)
